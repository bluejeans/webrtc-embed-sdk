Interface UIProps

Provides the object structure for the custom options supported for the UI modification.

param

by enabling this the SignIn option will be removed from the Haircheck screen

param

by enabling this the Full screen toggle button will be removed.

param

by enabling this the Meeting footer will be completely removed from InMeeting and PostMeeting screens.

param

by enabling this the chat panel/feature won't be provided

param

by enabling this the apps panel won't be provided.

param

by enabling this the Room Join option will be disabled.

param

by enabling this the Meeting controls will be locked always(No auto hide).

param

by enabling this the Meeting Link copy option will be removed from the Meeting page and the People panel.

param

by enabling this rating screen(QSore or TNPS) won't be shown to the end of the meeting.

param

by enabling this the no BlueJeans app pitches will be shown to the user.

param

this takes a string value, the color code or a gradient which can be applied as the background for the meeting screens.
