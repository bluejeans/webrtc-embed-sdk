Interface THSettings

Provides the object structure, for the customizations which can be provided for telehealth meetings. NOTE : When THSettings is provided, values for customBackground and inMeetingBGConfig will be ignored for a meeting. This is because we have a separate backgroundColor customisation option within THSetting meant for telehealth meetings.
